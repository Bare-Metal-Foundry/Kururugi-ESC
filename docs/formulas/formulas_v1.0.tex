\section{Design of the Voltage sensing}

As the current sensing uses an op-amp buffer with very low output impedance, the ADC input impedance constraint becomes negligible. Therefore, detailed RC settling analysis at the ADC pin is not required for the current sensing ADC channels. As the ADC of the STM32G484 are designed for this purpose, no additional impedance matching or damping network is deemed necessary.
\[
R_{V_1} = R_{V_2}(\frac{V_{ph,spike}}{V_{ADC}} - 1)
\]
\[
P_{R_{V_1}} = \frac{V_{ph,spike}^2}{R_{V_1}+R_{V_2}}
\]
\[
P_{R_{V_1},R_{V_2}} = \frac{V_{ph,spike}^2}{R_{V_1}+R_{V_2}}
\]
\[
V_{divider,spike} = \frac{R_{V_2}}{R_{V_1}+R_{V_2}} V_{ph,spike}
\]
\[
V_{divider,typ} = \frac{R_{V_2}}{R_{V_1}+R_{V_2}} V_{ph,typ}
\]
\[
I_{divider,typ} = \frac{V_{divider,typ}}{R_{V_1}+R_{V_2}}
\]
\[
I_{in,ADC,spike} = \frac{V_{divier,spike}}{R_{adc,ext}}
\]
\[
I_{in,ADC,typ} = \frac{V_{divier,typ}}{R_{adc,ext}}
\]
\[
R_{thvn} = \frac{R_{V_1}*R_{V_2}}{R_{V_1}+R_{V_2}}
\]
\[
I_{in,ADC,typ} = \frac{V_{divier,typ}}{R_{thvn}+R_{adc,ext}}
\]
\[
\Delta V_{LSB} = \frac{V_{ref,ADC}}{V_{in,ADC}(2^{N_{bit}}-1)}V_{ph,spike}
\]
\[
T_{aqc} = N_{ADC} * T_{ADC}
\]
To chose the external capacitor, one should note that we won't be sampling the ADC during blanking time. Hence, the external capacior will be fully charged at the start of the sampling: we can model the external capacitor as fully charge at the start of a sampling. This assumption holds when $t_{\mathrm{settle}} \gg R_{\mathrm{th}}\,C_{\mathrm{ext}}$.
\[
V_{err} = V_{divider,typ} e^{-\frac{t_{acq}}{R_{ADC,ext} C_{samp}}}
\]
\[
e^{-\frac{t_{acq}}{R_{ADC,ext} C_{samp}}} \le \frac{\epsilon_{LSB}}{2^N}
\]
\[
T_{fill,Csamp} = 
k_{charge}(R_{ADC,ext}+R_{thvn})C_{samp} \ln\left(\frac{2^{N_{bit}}}{\epsilon_{LSB}}\right)
